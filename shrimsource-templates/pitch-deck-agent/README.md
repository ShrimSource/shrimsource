# Pitch Deck Agent

Analyzes a startup idea and generates an outline for a compelling 10-slide pitch deck.

Run with Shrimsource:
```bash
npx create-shrimsource pitch-deck-agent
```

## Example

**Input:**
```
Idea: Uber for dog walking
```

**Output:**
```
Slide 1: Problem - Dogs are lonely, owners are busy.
Slide 2: Solution - On-demand trusted dog walkers...
Slide 10: Ask - Seeking $1M Seed to expand to 5 cities.
```

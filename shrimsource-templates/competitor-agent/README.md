# Competitor Agent

Analyze market position and top competitors for any company.

Run with Shrimsource:
```bash
npx create-shrimsource competitor-agent
```

## Example

**Input:**
```
openai.com
```

**Output:**
```
Competitors:
Anthropic
Google DeepMind
Cohere

Feature Comparison:
- Anthropic: Strong in enterprise, catching up on consumer AI
- Google DeepMind: Strong in enterprise, catching up on consumer AI
- Cohere: Strong in enterprise, catching up on consumer AI

Market Position:
openai.com remains a market leader with significant mindshare, but faces increasing pressure from well-funded alternatives.
```

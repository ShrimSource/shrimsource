# Market Map Agent

Input a market domain and generate a market map listing the top 10 companies by tier.

Run with Shrimsource:
```bash
npx create-shrimsource market-map-agent
```

## Example

**Input:**
```
AI coding tools
```

**Output:**
```
Market Map

Tier 1
- GitHub Copilot
- Cursor
- CodeWhisperer

Tier 2
- Codeium
- Tabnine
- Supermaven
```

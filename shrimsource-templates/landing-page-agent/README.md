# Landing Page Agent

Input a product idea and generate landing page copy including headlines, features, and a call-to-action.

Run with Shrimsource:
```bash
npx create-shrimsource landing-page-agent
```

## Example

**Input:**
```
AI code reviewer
```

**Output:**
```
Headline:
Review every PR with AI.

Features:
- PR analysis
- Bug detection
- Security scanning
```

# Resume Reviewer Agent

Analyzes a resume against a job description, provides a roast and actionable improvement tips.

Run with Shrimsource:
```bash
npx create-shrimsource resume-reviewer-agent
```

## Example

**Input:**
```
Resume: [Software Engineer...]
Job Description: [Senior React Developer...]
```

**Output:**
```
Review Report:
- Missing critical React Testing Library experience
- Roast: "You applied for a React role but only listed jQuery..."
- Tip: Add a bullet point highlighting your frontend state management.
```

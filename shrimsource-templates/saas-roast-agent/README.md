# SaaS Roast Agent

Input a website and output a product roast with weaknesses and growth ideas.

Run with Shrimsource:
```bash
npx create-shrimsource saas-roast-agent
```

## Example

**Input:**
```
notion.so
```

**Output:**
```
Product Roast

Strengths:
strong ecosystem
highly customizable

Weaknesses:
steep learning curve
can be slow with large databases

Growth Ideas:
AI onboarding assistant
simplified templates for beginners
```

# Social Media Agent

Generates a week of social media posts (X, LinkedIn) from a blog post or product URL.

Run with Shrimsource:
```bash
npx create-shrimsource social-media-agent
```

## Example

**Input:**
```
Topic: Launching our new open-source AI platform
```

**Output:**
```
Monday: 🚀 Big news! Just open sourced...
Wednesday: 🧵 Here is a thread on how we built it...
Friday: 💡 3 lessons learned from our launch day...
```

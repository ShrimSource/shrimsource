# Startup Idea Agent

Input a domain and generate 5 startup ideas with market analysis.

Run with Shrimsource:
```bash
npx create-shrimsource startup-idea-agent
```

## Example

**Input:**
```
AI tools
```

**Output:**
```
Startup Ideas:

1. AI Meeting Memory
Market: Remote work tools
Customers: startups, agencies

2. Prompt Analytics
Market: AI SaaS
Customers: developers
```

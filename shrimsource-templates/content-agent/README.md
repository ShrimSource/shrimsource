# Content Agent

Generate social media posts, blog ideas, and marketing copy.

Run with Shrimsource:
```bash
npx create-shrimsource content-agent
```

## Example

**Input:**
```
AI agent framework launch
```

**Output:**
```
Twitter Post:
🚀 Just launched our new AI agent framework! Build autonomous agents in < 100 lines of code.

LinkedIn Post:
We are thrilled to announce the launch of our new open-source AI agent framework...
```

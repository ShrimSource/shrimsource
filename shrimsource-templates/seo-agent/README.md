# SEO Agent

Generate blog outlines, SEO keywords, and article drafts.

Run with Shrimsource:
```bash
npx create-shrimsource seo-agent
```

## Example

**Input:**
```
AI coding tools
```

**Output:**
```
Title: Best AI Coding Tools in 2025

Outline:
1. Introduction
2. GitHub Copilot
3. Cursor
4. Conclusion

Are you looking for the best AI coding tools? You are in the right place. In this guide, we will explore the top tools...

Keywords: AI coding tools, free AI coding tools, best AI coding tools for developers
```

# Newsletter Agent

Curates recent news on a given topic and drafts a professional newsletter formatting.

Run with Shrimsource:
```bash
npx create-shrimsource newsletter-agent
```

## Example

**Input:**
```
Topic: AI Agent trends this week
```

**Output:**
```
Subject: The Agent Era is Here - Weekly Roundup

Welcome back! Here are the top 3 tools this week...
1. AutoGPT v2 Release...
2. Anthropic's new tool use API...
```

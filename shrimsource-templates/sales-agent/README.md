# Sales Agent

Generate personalized cold emails for growth teams.

Run with Shrimsource:
```bash
npx create-shrimsource sales-agent
```

## Example

**Input:**
```json
{
  "company": "notion.so",
  "product": "AI analytics platform"
}
```

**Output:**
```
Subject: Idea for Notion data infrastructure

Hi Notion team,

I noticed Notion recently launched new AI features last month. Huge congrats on the launch!

I'm reaching out because we help companies like yours leverage our AI analytics platform. We provide data-driven insights, improved conversion rates, and automated reporting.

Would you be open to a brief chat next week to see if there's a fit?

Best,
Sales Agent
```

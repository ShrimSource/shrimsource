# Cold Email Agent

Input a target company and your product offering to automatically gather company context and generate a highly personalized cold email.

Run with Shrimsource:
```bash
npx create-shrimsource cold-email-agent
```

## Example

**Input:**
```
Company: Shopify
Product: analytics dashboard
```

**Output:**
```
Subject:
Boost Shopify store analytics

Email:
Hi Shopify team...
```
